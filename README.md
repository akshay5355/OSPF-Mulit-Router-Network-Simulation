# Multi-Router OSPF Network Simulation

## Project Overview
This project demonstrates a multi-router network configured using the OSPF (Open Shortest Path First) routing protocol in Cisco Packet Tracer.

## Network Topology
- 3 Routers
- 3 Switches
- 9 PCs
- OSPF Dynamic Routing

## Features Implemented
- IP Address Configuration
- OSPF Routing Configuration
- Inter-Network Communication
- Connectivity Testing using Ping
- Routing Table Verification

## Tools Used
- Cisco Packet Tracer

## Outcome
All PCs across different networks successfully communicate through OSPF dynamic routing.
